# Анализ фильмов через TMDB API 

Проект по сбору и анализу данных о фильмах с использованием TMDB API.

## Что сделано
- Парсинг данных о фильмах через TMDB API (по названиям из датасета)
- Загрузка и хранение данных в PostgreSQL
- Создание интерактивного дашборда в Power BI

## Технологии
- Python + requests (парсинг API)
- PostgreSQL (хранение данных)
- Power BI (визуализация и аналитика)

## Дашборд
- В репозитории представлен файл `analitic_films.pbix` с готовым аналитическим отчётом.
- Если нет Power BI можно посмотреть дашборд в файле 'analitic_films.pdf'
